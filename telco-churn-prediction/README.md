# Telco Customer Churn Prediction

## Project Overview
This project builds predictive models to identify customers at high risk of churning (leaving) in a telecommunications company. The objective is to enable proactive retention strategies and reduce revenue loss.

## Key Results
- **Best Model**: Random Forest classifier
- **F1-Score**: 0.6374 (after hyperparameter tuning)
- **ROC-AUC**: ~0.84
- **Key Churn Drivers**: Month-to-month contracts, low tenure, lack of tech support / online security, electronic check payments

## Files in this folder
- `Telco_Churn_Prediction_Notebook.ipynb` – End-to-end analysis and modeling
- `Telco_Churn_Project_Report.docx` – Comprehensive project report with business recommendations

## Business Impact
The model supports targeted retention campaigns that can significantly reduce churn and deliver strong ROI through improved customer retention.
