# AIML Portfolio Projects

## 🚀 Red Wine Quality Prediction (Vino Veritas)
**Predicting wine quality from physicochemical properties using ML classification & regression.**

- **Notebooks**: `1. wine_quality_ml_models.ipynb`, `2. wine_quality_conclusions.ipynb`, `summary_notebook.ipynb`
- **Data**: `winequality_red_cleansed.csv`
- **Reports**: `Wine_Quality_Project_Report.docx`
- **Key Results**: Tuned LightGBM (88.8% multi-class accuracy), XGBoost (90.4% binary)
- **Top Features**: Alcohol, Sulphates, Volatile Acidity

## 📞 Telco Customer Churn Prediction
**Predicting customer attrition in telecommunications using ensemble ML models.**

- **Notebook**: `Telco_Churn_Prediction_Notebook.ipynb`
- **Report**: `Telco_Churn_Project_Report.docx`
- **Key Results**: Random Forest (F1-Score 0.6374), ROC-AUC ~0.84
- **Top Churn Drivers**: Month-to-month contracts, low tenure, lack of tech support
